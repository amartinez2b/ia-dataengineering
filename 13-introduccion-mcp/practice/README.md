# Práctica - Módulo 13

## Objetivo práctico

Instalar y usar MCPs en **Codex** y **Claude Code** para resolver dos tareas concretas del curso:

- diseñar una arquitectura técnica y diagramarla con draw.io
- leer tablas de una base MySQL y generar un modelo de datos

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: instalar la skill `architecture-diagram-generator` en Codex
- Actividad 2: instalar el MCP de draw.io en Codex
- Actividad 3: diseñar una arquitectura con la skill y diagramarla en draw.io desde Codex
- Actividad 4: repetir el flujo anterior en Claude Code
- Actividad 5: instalar el MCP de MySQL en Codex
- Actividad 6: inspeccionar tablas reales y generar un modelo de datos desde Codex
- Actividad 7: repetir el flujo anterior en Claude Code
- Actividad 8: comparar resultados y revisar calidad técnica

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- instalar un servidor MCP en Codex
- instalar un servidor MCP en Claude Code
- instalar y usar una skill complementaria dentro del flujo
- usar draw.io MCP para abrir un diagrama técnico
- usar MySQL MCP para inspeccionar tablas reales
- producir un diagrama de arquitectura y un modelo de datos
- distinguir entre observación real e inferencia del agente
