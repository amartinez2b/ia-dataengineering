# Práctica - Módulo 01

## Objetivo práctico

Comprender cómo un LLM puede asistir tareas reales de desarrollo en ingeniería de datos y aprender a detectar respuestas problemáticas o alucinadas en una herramienta de uso cotidiano como ChatGPT.

## Modalidad sugerida

Laboratorio guiado en parejas o individual, usando ChatGPT como herramienta principal.

## Actividades

- Actividad 1: crear o validar acceso a una plataforma de IA
- Actividad 2: ejecutar un prompt sencillo
- Actividad 3: comparar una respuesta útil con una respuesta alucinada o con inferencias injustificadas

## Material de esta sección

- [Ejercicio 01 - Prompt y alucinaciones](./ejercicio-01-prompt-y-alucinaciones.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder distinguir entre:

- una respuesta útil del LLM
- una respuesta incorrecta o alucinada
- una respuesta basada en evidencia y una respuesta con suposiciones
- una demostración convincente y una validación técnica real

La meta no es aceptar lo que diga la IA, sino aprender a trabajar con ella con criterio técnico.
