# Módulo 01 - Introducción a la IA para ingenieros de datos

## Objetivo del módulo

Comprender qué son la IA, el machine learning y los LLMs, y familiarizarse con las plataformas de IA generativa más usadas en el mercado para comenzar a trabajar con ellas desde el primer día del curso.

## Enfoque del módulo

Este módulo tiene una pequeña base conceptual y un componente práctico dominante. La meta no es estudiar teoría profunda de IA, sino construir un criterio útil para trabajar con asistentes y agentes en contextos reales de data engineering.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- Diferenciar IA, ML, deep learning y LLMs.
- Explicar a alto nivel cómo funciona un LLM.
- Identificar capacidades, limitaciones y riesgos de los LLMs.
- Reconocer las principales plataformas de IA generativa usadas hoy: Google, OpenAI, Anthropic y DeepSeek.
- Crear cuentas y acceder a estas herramientas para comenzar las prácticas del curso.

## Manual rápido - Crear cuentas y preparar el entorno

Esta guía está pensada para que cada estudiante llegue a la práctica con acceso a las plataformas más usadas en el curso y con su copia del repositorio lista para trabajar.

### 1. GitHub

Abre [GitHub](https://github.com/).

#### Crear una cuenta

- entra al sitio oficial
- selecciona la opción de registro
- crea la cuenta con correo, usuario y contraseña
- verifica el correo electrónico si GitHub lo solicita
- inicia sesión y confirma que puedes acceder a tu perfil

#### Hacer fork del repositorio del curso

Una vez dentro de GitHub:

1. Abre el repositorio original del curso: [amartinez2b/ia-dataengineering](https://github.com/amartinez2b/ia-dataengineering)
2. Haz clic en `Fork`
3. Crea una copia en tu propia cuenta

El objetivo es que cada estudiante tenga su propia copia del repositorio para trabajar durante el curso.

#### Clonar el repositorio a la computadora

Después de hacer el fork, entra a tu copia del repositorio y usa la URL de tu cuenta.

Ejemplo:

```bash
git clone https://github.com/TU-USUARIO/ia-dataengineering.git
```

Luego entra a la carpeta:

```bash
cd ia-dataengineering
```

### 2. OpenAI - ChatGPT

Abre [ChatGPT](https://chatgpt.com/).

#### Pasos sugeridos

- entra al sitio oficial
- selecciona crear cuenta o iniciar sesión
- continúa con correo, Google, Microsoft o Apple
- completa la verificación si el sistema la solicita
- abre un chat nuevo y valida que puedas escribir prompts

### 3. Google - Gemini

Abre [Gemini](https://gemini.google.com/).

#### Pasos sugeridos

- inicia sesión con una cuenta de Google personal o de trabajo compatible
- acepta los términos si es la primera vez que ingresas
- valida que puedes abrir la interfaz y empezar una conversación
- revisa que la cuenta quede disponible para futuras clases

### 4. Anthropic - Claude

Abre [Claude](https://claude.ai/).

#### Pasos sugeridos

- selecciona la opción de registro o inicio de sesión
- usa el método disponible en tu región, normalmente correo o cuenta vinculada
- completa la verificación solicitada por la plataforma
- entra a la interfaz y confirma que puedes iniciar un chat

### 5. DeepSeek

Abre [DeepSeek Chat](https://chat.deepseek.com/).

#### Pasos sugeridos

- entra al sitio oficial
- crea una cuenta o inicia sesión con el método disponible
- verifica el correo si es necesario
- entra al chat y prueba una instrucción simple

### Checklist de validación para clase

Antes de comenzar la práctica, cada estudiante debería confirmar que puede:

- entrar a su cuenta de GitHub
- ver su fork del repositorio del curso
- clonar el repositorio localmente
- abrir al menos una de estas plataformas sin problemas
- crear una conversación nueva
- escribir un prompt
- copiar y pegar texto o fragmentos de código
- volver a conversaciones anteriores

### Recomendaciones para la clase

- Haz el fork antes de clonar, para que el repositorio quede asociado a tu propia cuenta.
- Usa una cuenta personal o académica a la que tengas acceso durante toda la capacitación.
- No compartas contraseñas ni credenciales con otras personas.
- No pegues secretos, tokens, contraseñas o datos sensibles en el chat.
- Si la interfaz cambia, mantén el mismo objetivo: crear cuenta, iniciar sesión y abrir un chat nuevo.
- No todas las plataformas ofrecen exactamente el mismo flujo según país, edad o tipo de cuenta.

### Nota para el instructor

Si algún estudiante no logra registrarse en una de las plataformas, puede trabajar con otra de la lista o en pareja mientras resuelve el acceso.

## Agenda sugerida para 1.5 horas

- 15 min: conceptos base de IA, ML y LLMs
- 15 min: cómo funcionan los LLMs y qué pueden hacer bien
- 15 min: limitaciones, riesgos y buenas prácticas
- 35 min: laboratorio guiado
- 10 min: discusión de resultados

## Mensaje clave

Un LLM no reemplaza el criterio de un ingeniero de datos. Su mayor valor está en acelerar análisis, documentación, generación de borradores, debugging y exploración de soluciones, siempre con validación humana.
