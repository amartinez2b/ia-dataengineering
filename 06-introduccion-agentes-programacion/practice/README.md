# Práctica - Módulo 06

## Objetivo práctico

Instalar y configurar agentes de programación reales, autenticados con API key, y usarlos para construir un proyecto web simple dentro de una carpeta local.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: instalar Claude Code
- Actividad 2: configurar autenticación con API key
- Actividad 3: crear una carpeta de proyecto en Windows
- Actividad 4: usar Claude Code para generar una página web turística
- Actividad 5: enriquecer el prompt para mejorar el resultado final
- Actividad 6: repetir el flujo con Codex y comparar la experiencia

## Material de esta sección

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- instalar y lanzar Claude Code
- instalar y lanzar Codex
- configurar autenticación con API key
- usar un agente dentro de una carpeta real
- mejorar un prompt para obtener una página web más funcional y atractiva
- comparar cómo cambia la experiencia al usar dos agentes distintos
