# Módulo 09 - Desarrollo práctico agentico

## Objetivo del módulo

Aprender a usar agentes como Claude Code y Codex en un flujo de desarrollo real, incluyendo cómo pedir tareas complejas, cómo iterar sobre cambios, cómo apoyar refactorizaciones y cómo usar el agente para debugging de forma responsable.

## Enfoque del módulo

Este módulo es principalmente práctico. La teoría existe para dar marco, pero la meta es que el estudiante aprenda a trabajar con Claude Code y Codex dentro de un proyecto real como si fueran colaboradores técnicos.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- usar Claude Code y Codex en tareas concretas de desarrollo
- formular tareas complejas de forma clara
- pedir refactorizaciones asistidas
- usar agentes para analizar errores y debugging
- iterar cambios con criterio técnico

## Agenda sugerida para 1.5 horas

- 15 min: flujo práctico de trabajo con Claude Code y Codex
- 20 min: cómo pedir tareas complejas
- 20 min: refactorización y debugging asistido
- 25 min: ejercicio práctico guiado
- 10 min: cierre y aprendizajes

## Mensaje clave

Un agente funciona mejor cuando se le trata como un colaborador técnico con contexto, límites y validación, no como una caja mágica a la que se le delega todo sin revisar.
