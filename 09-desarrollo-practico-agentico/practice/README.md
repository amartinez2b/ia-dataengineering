# Práctica - Módulo 09

## Objetivo práctico

Usar Claude Code y Codex para analizar, refactorizar y depurar una pequeña tarea técnica dentro de un proyecto real o de ejemplo.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: pedir análisis inicial de una tarea compleja
- Actividad 2: usar Claude Code para proponer una mejora o refactor
- Actividad 3: usar Claude Code para diagnosticar un error
- Actividad 4: repetir el flujo de análisis, refactor y debugging con Codex
- Actividad 5: comparar la experiencia entre terminal e interfaz visual
- Actividad 6: validar qué cambió y qué sigue bajo responsabilidad humana

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- formular una tarea compleja para Claude Code y Codex
- usar el agente para analizar y proponer cambios
- usar el agente para explicar un error
- comparar diferencias de flujo entre ambos agentes
- validar qué partes de la solución requieren revisión humana
