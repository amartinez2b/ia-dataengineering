# Práctica - Módulo 09

## Objetivo práctico

Usar Claude Code para analizar, refactorizar y depurar una pequeña tarea técnica dentro de un proyecto real o de ejemplo.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: pedir análisis inicial de una tarea compleja
- Actividad 2: usar Claude Code para proponer una mejora o refactor
- Actividad 3: usar Claude Code para diagnosticar un error
- Actividad 4: validar qué cambió y qué sigue bajo responsabilidad humana

## Material de esta sección

- [Ejercicio 01 - Refactorización y debugging con Claude Code](./ejercicio-01-refactor-debug-claude-code.md)
- [Ejercicio 02 - Usar Claude Code dentro de Visual Studio Code](./ejercicio-02-claude-code-vscode.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- formular una tarea compleja para Claude Code
- usar el agente para analizar y proponer cambios
- usar el agente para explicar un error
- validar qué partes de la solución requieren revisión humana
