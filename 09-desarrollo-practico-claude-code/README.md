# Módulo 09 - Desarrollo práctico con Claude Code

## Objetivo del módulo

Aprender a usar Claude Code en un flujo de desarrollo real, incluyendo cómo pedir tareas complejas, cómo iterar sobre cambios, cómo apoyar refactorizaciones y cómo usar el agente para debugging de forma responsable.

## Enfoque del módulo

Este módulo es principalmente práctico. La teoría existe para dar marco, pero la meta es que el estudiante aprenda a trabajar con Claude Code dentro de un proyecto real como si fuera un colaborador técnico.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- usar Claude Code en tareas concretas de desarrollo
- formular tareas complejas de forma clara
- pedir refactorizaciones asistidas
- usar Claude Code para analizar errores y debugging
- iterar cambios con criterio técnico

## Agenda sugerida para 1.5 horas

- 15 min: flujo práctico de trabajo con Claude Code
- 20 min: cómo pedir tareas complejas
- 20 min: refactorización y debugging asistido
- 25 min: ejercicio práctico guiado
- 10 min: cierre y aprendizajes

## Mensaje clave

Claude Code funciona mejor cuando se le trata como un colaborador técnico con contexto, límites y validación, no como una caja mágica a la que se le delega todo sin revisar.
