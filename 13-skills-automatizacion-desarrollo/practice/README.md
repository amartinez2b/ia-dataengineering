# Práctica - Módulo 13

## Objetivo práctico

Instalar y usar la skill `docx` para generar un documento Word con la documentación técnica del proyecto de datos construido en los temas 11 y 12.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: instalar las document skills en Claude Code
- Actividad 2: preparar la información técnica del proyecto
- Actividad 3: usar la skill `docx` para generar un `.docx`
- Actividad 4: revisar el documento final y validar su calidad

## Material de esta sección

- [Ejercicio 01 - Generar un documento Word técnico con la skill docx](./ejercicio-01-documentacion-word-con-docx-skill.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- instalar una skill especializada en Claude Code
- usar la skill `docx` para crear un documento Word
- documentar técnicamente el pipeline y el DAG del proyecto
- obtener un entregable profesional reutilizable
