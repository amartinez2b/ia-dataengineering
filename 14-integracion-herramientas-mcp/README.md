# Contenido integrado en el Módulo 13

El contenido que antes estaba separado en este directorio fue integrado dentro del módulo:

- [Módulo 13 - MCP para diagramación y modelado de datos](../13-introduccion-mcp/README.md)

La idea del curso ahora es trabajar MCP como un solo bloque práctico que combine:

- diagramación técnica con draw.io
- diseño de arquitectura con la skill `architecture-diagram-generator`

- generación de un modelo de datos
