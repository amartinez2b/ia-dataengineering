# Módulo 15 - Integración de herramientas mediante MCP

## Objetivo del módulo

Integrar un agente con una base de datos real mediante MCP para inspeccionar esquemas, consultar tablas y generar documentación técnica útil para un proyecto de ingeniería de datos.

## Enfoque del módulo

Este módulo es principalmente práctico. La teoría introduce cómo MCP permite conectar agentes con herramientas reales de trabajo. La práctica se enfoca en instalar un MCP para MySQL, conectarse a una base de datos remota y generar un catálogo de datos.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- explicar cómo un agente usa MCP para conectarse a una base de datos
- instalar un servidor MCP para MySQL
- configurar variables de entorno para una conexión segura
- inspeccionar tablas y esquemas con ayuda de IA
- generar documentación técnica tipo catálogo de datos

## Agenda sugerida para 1.5 horas

- 15 min: MCP aplicado a herramientas de datos
- 15 min: conexión a MySQL vía MCP
- 20 min: exploración de esquemas y tablas
- 30 min: generación del catálogo de datos
- 10 min: revisión final

## Mensaje clave

La verdadera potencia de MCP aparece cuando el agente deja de trabajar solo con texto y puede interactuar con herramientas reales. En este módulo, esa integración se usa para convertir una base de datos viva en documentación técnica estructurada.
