# Práctica - Módulo 04

## Objetivo práctico

Diseñar, probar y mejorar prompts para obtener mejores respuestas técnicas al pedir explicaciones, código Python y transformaciones PySpark.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: detectar problemas en prompts débiles
- Actividad 2: mejorar prompts con una estructura más clara
- Actividad 3: comparar resultados entre un prompt débil y uno bien diseñado
- Actividad 4: iterar un prompt hasta obtener una respuesta más útil

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- identificar qué le falta a un prompt
- mejorar una instrucción técnica
- pedir una salida más estructurada
- justificar por qué una segunda versión del prompt funciona mejor que la primera
