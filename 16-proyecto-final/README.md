# Módulo 17 - Proyecto final

## Objetivo del módulo

Aplicar los conceptos del curso en un reto final completo que combine conexión a base de datos, ingeniería de datos, construcción de un dataset analítico, desarrollo de un dashboard HTML y publicación de resultados en un repositorio público.

## Enfoque del módulo

Este módulo es totalmente práctico. La teoría solo entrega un marco mínimo para estructurar el reto. Todo el valor del módulo está en ejecutar el proyecto de punta a punta y presentar una entrega final pública y reproducible.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- conectarse a una base de datos real y extraer tablas de negocio
- construir un dataset integrado llamado `comercial`
- almacenar resultados en formato Parquet
- desarrollar un dashboard en HTML con visualizaciones de negocio
- organizar un repositorio final con código, documentación y archivos de contexto de IA
- publicar y compartir una solución final de ingeniería de datos asistida por IA

## Agenda sugerida para 1.5 horas

- 10 min: revisión del reto y entregables
- 20 min: ingesta y construcción del dataset `comercial`
- 20 min: generación del dashboard HTML
- 20 min: organización del repositorio final
- 20 min: publicación y entrega

## Mensaje clave

El proyecto final no mide solo si el estudiante sabe usar una herramienta puntual. Mide si puede integrar datos, transformación, visualización, documentación y colaboración con IA en una entrega técnica completa y presentable.
