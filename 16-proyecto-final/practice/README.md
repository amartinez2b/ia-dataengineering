# Práctica - Módulo 17

## Objetivo práctico

Construir un proyecto final completo de ingeniería de datos asistida por IA, desde la ingesta hasta la visualización final y su publicación en GitHub.

## Modalidad sugerida

Trabajo individual.

## Actividades

- Actividad 1: conectarse a la base de datos `fake`
- Actividad 2: ingerir `customers`, `products`, `sales` y `shops`
- Actividad 3: construir el dataset `comercial`
- Actividad 4: guardar el resultado en Parquet
- Actividad 5: construir el dashboard HTML
- Actividad 6: organizar y publicar el repositorio `ia-challenge`
- Actividad 7: enviar el enlace final por correo

## Material de esta sección

- [Ejercicio 01 - Challenge final de IA Data Engineering](./ejercicio-01-challenge-final-ia-data-engineering.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- construir un pipeline funcional de integración de datos
- generar un dataset analítico único
- crear un dashboard HTML funcional
- publicar un repositorio completo y bien documentado
