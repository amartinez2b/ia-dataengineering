# Práctica - Módulo 15

## Objetivo práctico

Instalar un MCP para MySQL, conectarse a una base de datos real y generar un catálogo técnico de las tablas usando un agente de IA.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: instalar el MCP de MySQL
- Actividad 2: configurar la conexión a la base `fake`
- Actividad 3: explorar tablas y esquema con el agente
- Actividad 4: generar un catálogo de datos
- Actividad 5: revisar y mejorar la documentación resultante

## Material de esta sección

- [Ejercicio 01 - Instalar MCP de MySQL y generar un catálogo de datos](./ejercicio-01-mysql-mcp-y-catalogo-datos.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- instalar un servidor MCP de base de datos
- conectarse a MySQL desde Claude Code
- inspeccionar tablas con ayuda de IA
- producir un catálogo de datos legible y útil
