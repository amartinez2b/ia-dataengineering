# Práctica - Módulo 15

## Objetivo práctico

Instalar un MCP para MySQL en Claude Code y OpenCode, conectarse a una base de datos real y generar un catálogo técnico de las tablas usando un agente de IA.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: instalar el MCP de MySQL en Claude Code
- Actividad 2: configurar la conexión a la base `fake`
- Actividad 3: repetir la integración en OpenCode
- Actividad 4: explorar tablas y esquema con el agente
- Actividad 5: generar un catálogo de datos
- Actividad 6: revisar y mejorar la documentación resultante

## Material de esta sección

- [Ejercicio 01 - Instalar MCP de MySQL y generar un catálogo de datos](./ejercicio-01-mysql-mcp-y-catalogo-datos.md)
- [Ejercicio 02 - Configurar MCP de MySQL en OpenCode y generar un catálogo de datos](./ejercicio-02-mysql-mcp-en-opencode.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- instalar un servidor MCP de base de datos
- instalar un servidor MCP de base de datos en OpenCode
- conectarse a MySQL desde Claude Code
- conectarse a MySQL desde OpenCode
- inspeccionar tablas con ayuda de IA
- producir un catálogo de datos legible y útil
