# Práctica - Módulo 02

## Objetivo práctico

Explorar cómo cambia el uso de un modelo cuando miramos su contexto, sus tokens, su tamaño y las herramientas disponibles para entenderlo mejor.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: medir cómo crece el prompt en tokens
- Actividad 2: observar herramientas visuales para entender ML y LLMs
- Actividad 3: discutir qué tipo de modelo convendría para distintos escenarios

## Material de esta sección

- [Ejercicio 01 - Conteo de tokens](./ejercicio-01-conteo-de-tokens.md)
- [Ejercicio 02 - Visualización de ML y LLMs](./ejercicio-02-visualizacion-ml-llm.md)
- [Ejercicio 03 - Comparación de precios de LLMs](./ejercicio-03-comparacion-precios-llms.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- interpretar qué significa un conteo de tokens
- distinguir tokens de parámetros
- usar herramientas visuales para reforzar la intuición sobre ML y LLMs
- argumentar qué modelo elegiría según un caso de uso simple
