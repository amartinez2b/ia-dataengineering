# Módulo 02 - Ecosistema actual de LLMs

## Objetivo del módulo

Conocer el ecosistema actual de modelos de lenguaje, diferenciar alternativas propietarias y open source, y entender criterios prácticos para elegir un modelo según costo, capacidad, privacidad, contexto, tamaño y entorno de ejecución.

## Enfoque del módulo

Este módulo combina una base conceptual breve con ejercicios prácticos para observar diferencias reales entre modelos, contexto en tokens y herramientas visuales que ayudan a explicar cómo funcionan.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- Diferenciar LLMs propietarios y open source.
- Reconocer actores importantes del ecosistema actual.
- Entender qué significa comparar modelos por capacidad, tamaño, costo y contexto.
- Relacionar tokens, parámetros y memoria de ejecución.
- Elegir un modelo adecuado para una tarea concreta de ingeniería de datos.

## Agenda sugerida para 1.5 horas

- 20 min: mapa del ecosistema actual de LLMs
- 20 min: comparación entre modelos propietarios y open source
- 15 min: tokens, contexto, parámetros y memoria
- 25 min: ejercicios prácticos
- 10 min: discusión de cuándo usar cada tipo de modelo

## Mensaje clave

No existe un LLM perfecto para todo. La elección correcta depende de la tarea, del presupuesto, de la privacidad, del entorno técnico y del nivel de calidad que se necesita.
