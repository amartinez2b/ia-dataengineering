# Práctica - Módulo 11

## Objetivo práctico

Extender un proyecto real de ingeniería de datos usando agentes de IA para construir un flujo `raw -> bronze -> silver -> gold` sobre varias tablas de negocio.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: copiar el repositorio base a una cuenta personal
- Actividad 2: crear la rama `dev` y clonar el proyecto localmente
- Actividad 3: analizar el repositorio con un agente de IA
- Actividad 4: desarrollar la ingesta `raw -> bronze`
- Actividad 5: desarrollar limpieza `bronze -> silver`
- Actividad 6: consolidar una tabla `comercial` en `gold`

## Material de esta sección

- [Ejercicio 01 - Pipeline PySpark con agentes de IA sobre multihope](./ejercicio-01-pipeline-pyspark-con-agentes.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- trabajar sobre una copia propia del repositorio
- desarrollar en una rama `dev`
- usar agentes de IA para analizar y extender un proyecto de datos
- implementar tablas `bronze`, `silver` y `gold`
- justificar las decisiones técnicas tomadas durante el desarrollo
