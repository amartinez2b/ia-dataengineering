# Práctica - Módulo 05

## Objetivo práctico

Practicar una metodología real de trabajo asistido por IA analizando un repositorio existente, detectando problemas funcionales y de seguridad, y usando la IA para acelerar investigación y diagnóstico.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: clonar un repositorio real y analizar su estructura con IA
- Actividad 2: detectar fallas de funcionamiento
- Actividad 3: hacer revisión básica de vulnerabilidades
- Actividad 4: ejecutar notebooks y usar IA para diagnosticar errores
- Actividad 5: proponer una mejora adicional sobre el repositorio

## Material de esta sección

- [Ejercicio 01 - Analizar un repositorio real con ayuda de IA](./ejercicio-01-analisis-repositorio-con-ia.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- usar IA para explorar y entender un repositorio real
- identificar fallas de funcionamiento y riesgos básicos
- usar IA para diagnosticar errores de ejecución
- distinguir entre ayuda útil de la IA y validación manual obligatoria
