# Práctica - Módulo 03

## Objetivo práctico

Ejecutar un flujo básico de trabajo con Git y GitHub usando el repositorio real del proyecto **Multihope**, entendiendo cómo hacer cambios con seguridad y cómo preparar un proyecto de data engineering para colaborar con IA.

## Repositorio de trabajo

```
https://github.com/amartinez2b/multihope.git
```

Este repositorio contiene un pipeline de datos real con arquitectura Medallion (RAW → BRONZE → SILVER → GOLD) construido con PySpark, incluyendo un catálogo de datos diseñado para ser consumido por IA.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: clonar el repositorio y configurar credenciales de GitHub
- Actividad 2: revisar el estado del repositorio y entender su estructura
- Actividad 3: crear una rama de trabajo personal
- Actividad 4: hacer un cambio concreto en el proyecto y registrarlo con commit
- Actividad 5: analizar cómo está preparado el repositorio para trabajar con IA

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- identificar en qué rama está trabajando
- revisar cambios con `git status`
- crear un commit con mensaje claro
- explicar por qué conviene usar ramas y commits pequeños al trabajar con IA
