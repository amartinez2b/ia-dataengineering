# Módulo 03 - Git y GitHub para trabajo con IA

## Objetivo del módulo

Comprender los fundamentos de Git y GitHub, practicar el flujo básico de trabajo con repositorios y aprender buenas prácticas para colaborar con asistentes y agentes de IA en proyectos de software y data engineering.

## Enfoque del módulo

Este módulo combina teoría breve con una práctica muy operativa. La meta es que el estudiante pierda el miedo a Git, entienda el valor de GitHub y adopte un flujo ordenado para trabajar con IA sin perder control sobre el código.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- entender qué problema resuelve Git
- crear y clonar repositorios
- hacer cambios, commits y pushes básicos
- trabajar con ramas
- diferenciar repositorio local y remoto
- adoptar buenas prácticas para usar IA dentro de un repositorio real

## Agenda sugerida para 1.5 horas

- 20 min: fundamentos de Git y GitHub
- 20 min: commits, ramas y repositorios remotos
- 15 min: flujo de trabajo con IA
- 25 min: ejercicios prácticos
- 10 min: cierre y resolución de dudas

## Mensaje clave

Trabajar con IA sin Git es riesgoso. Git da trazabilidad, control de cambios, reversibilidad y una forma segura de experimentar con asistentes y agentes sin perder el trabajo.
