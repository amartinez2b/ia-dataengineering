# Teoría - Módulo 10

## 1. ¿Por qué un proyecto web simple?

Un proyecto web pequeño es una buena forma de practicar desarrollo asistido porque obliga a coordinar varios elementos al mismo tiempo:

- estructura de carpetas
- contenido
- diseño visual
- comportamiento básico
- revisión en navegador

## 2. Estructura mínima de un sitio web simple

Para este módulo, una estructura básica suficiente podría ser:

- `index.html`
- `styles.css`
- `script.js`

En algunos casos también podrían aparecer:

- carpeta `assets/`
- imágenes
- íconos

## 3. Qué puede hacer Claude Code en este tipo de proyecto

Claude Code puede ayudar a:

- crear los archivos iniciales
- proponer estructura HTML
- escribir estilos CSS
- agregar interacciones sencillas en JavaScript
- mejorar el contenido
- refinar diseño y responsividad

## 4. Qué sigue siendo responsabilidad del estudiante

Aunque Claude Code acelera mucho, el estudiante debe seguir validando:

- si la página realmente abre
- si el contenido tiene sentido
- si el diseño se ve bien
- si la navegación funciona
- si el resultado responde al objetivo del ejercicio

## 5. Flujo recomendado

Un flujo útil para este módulo es:

1. crear una carpeta vacía
2. pedir una primera versión
3. abrir la página
4. detectar mejoras
5. refinar el prompt
6. iterar hasta obtener un resultado mejor

## 6. Qué hace bueno a un prompt para proyecto web

Conviene indicar:

- objetivo del sitio
- público objetivo
- secciones requeridas
- estilo visual esperado
- restricciones técnicas
- formato o estructura de archivos

## 7. Qué se aprende realmente en este módulo

Más allá de construir una web, este módulo enseña:

- cómo trabajar con múltiples archivos
- cómo iterar con un agente
- cómo revisar resultados visuales
- cómo convertir una idea simple en una entrega concreta

## 8. Ideas clave para llevarse

- una página pequeña ya es suficiente para practicar coordinación entre archivos
- Claude Code ayuda a acelerar el borrador inicial
- el valor del estudiante está en revisar, ajustar y mejorar
- iterar el prompt cambia mucho la calidad del resultado final
