# Módulo 10 - Proyecto práctico 1: desarrollo web simple

## Objetivo del módulo

Construir desde cero una página web simple usando Claude Code, aprendiendo a coordinar múltiples archivos, iterar sobre el diseño y trabajar con un agente dentro de un proyecto real.

## Enfoque del módulo

Este módulo es principalmente práctico. La teoría solo introduce una estructura mínima para organizar un proyecto web pequeño. El valor principal está en usar Claude Code para crear, mejorar y ajustar una página web funcional.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- crear un proyecto web simple desde cero
- usar Claude Code para generar varios archivos coordinados
- iterar sobre contenido, diseño y estructura
- abrir y validar la página en el navegador
- refinar prompts para lograr un resultado más funcional y atractivo

## Agenda sugerida para 1.5 horas

- 15 min: estructura mínima de un proyecto web simple
- 15 min: cómo pedir a Claude Code una primera versión
- 20 min: iteración sobre diseño y contenido
- 30 min: construcción del proyecto
- 10 min: revisión final en navegador

## Mensaje clave

Un proyecto web pequeño es ideal para aprender a coordinar varios archivos con un agente. La clave no está solo en que Claude Code genere HTML, CSS y JS, sino en cómo vamos refinando el proyecto hasta lograr una página útil y atractiva.
