# Práctica - Módulo 10

## Objetivo práctico

Construir desde cero una página web que promocione turísticamente al Ecuador usando Claude Code, ya sea desde consola o desde el plugin de Visual Studio Code.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: crear una carpeta de proyecto vacía
- Actividad 2: pedir a Claude Code una primera versión del sitio
- Actividad 3: enriquecer el prompt para mejorar contenido y diseño
- Actividad 4: abrir la web en el navegador y validar el resultado

## Material de esta sección

- [Ejercicio 01 - Construir una web turística de Ecuador con Claude Code](./ejercicio-01-web-ecuador-claude-code.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- crear un proyecto web simple desde cero
- usar Claude Code para generar varios archivos
- iterar sobre el prompt para mejorar la página
- abrir el sitio en el navegador y revisar el resultado final
