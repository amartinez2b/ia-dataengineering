# Práctica - Módulo 07

## Objetivo práctico

Instalar, autenticar y probar tres agentes de programación CLI del ecosistema actual.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: instalar Codex y autenticar con API key de OpenAI
- Actividad 2: crear una API key en Google AI Studio, instalar Gemini CLI y autenticar
- Actividad 3: instalar OpenCode y autenticar con API key de OpenAI

## Material de esta sección

- [Ejercicio 01 - Instalar Codex y autenticar con API key](./ejercicio-01-codex.md)
- [Ejercicio 02 - Instalar Gemini CLI y autenticar con AI Studio](./ejercicio-02-gemini-cli.md)
- [Ejercicio 03 - Instalar OpenCode y autenticar con OpenAI](./ejercicio-03-opencode.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- instalar cada CLI
- cargar su API key correctamente
- iniciar cada herramienta al menos una vez
- describir diferencias básicas entre las tres experiencias
