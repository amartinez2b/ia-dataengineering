# Módulo 07 - Agentes Top

## Objetivo del módulo

Instalar, configurar y probar algunos de los agentes de programación más relevantes del ecosistema actual, entendiendo cómo autenticarlos con API keys y cómo integrarlos en un flujo de trabajo práctico desde terminal.

## Enfoque del módulo

Este módulo es casi totalmente práctico. La idea es que el estudiante no solo conozca los nombres de Codex, Gemini CLI y OpenCode, sino que pueda instalarlos, autenticarlos y usarlos como herramientas reales de trabajo.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- instalar Codex CLI, Gemini CLI y OpenCode
- autenticarse con API keys en cada herramienta
- entender diferencias de experiencia entre distintos agentes CLI
- usar cada herramienta dentro de un proyecto local
- comparar fortalezas y limitaciones de distintos agentes

## Mensaje clave

Conocer agentes “top” no significa memorizar marcas. Significa entender cómo instalarlos, autenticarlos y elegir el más adecuado para una tarea concreta.
