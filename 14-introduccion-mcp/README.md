# Módulo 14 - Introducción a MCP

## Objetivo del módulo

Entender qué es MCP, cómo permite conectar agentes con herramientas externas y cómo usarlo para extender las capacidades de un flujo de desarrollo asistido por IA.

## Enfoque del módulo

Este módulo es principalmente práctico. La teoría introduce la idea de MCP y sus componentes principales. La práctica se enfoca en instalar el MCP oficial de draw.io y usarlo para generar un diagrama del flujo de datos orquestado en el tema 12.

## Resultados de aprendizaje

Al finalizar este módulo, el estudiante podrá:

- explicar qué es MCP y para qué sirve
- distinguir entre agente, cliente MCP y servidor MCP
- instalar un servidor MCP en Claude Code
- invocar herramientas MCP desde el agente
- usar MCP para generar un diagrama técnico de un pipeline de datos

## Agenda sugerida para 1.5 horas

- 15 min: introducción conceptual a MCP
- 15 min: arquitectura cliente-servidor y herramientas
- 20 min: instalación del MCP de draw.io
- 30 min: generación del diagrama del DAG del tema 12
- 10 min: revisión y cierre

## Mensaje clave

MCP convierte a los agentes en sistemas más útiles porque les permite interactuar con herramientas externas de forma estructurada. En lugar de limitarse a conversar o generar texto, un agente con MCP puede crear diagramas, leer recursos y operar sobre herramientas reales.
