# Práctica - Módulo 14

## Objetivo práctico

Instalar el MCP oficial de draw.io en Claude Code y usarlo para generar un diagrama visual del flujo de datos orquestado en el tema 12.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: instalar el MCP de draw.io
- Actividad 2: validar que Claude Code conozca las herramientas del MCP
- Actividad 3: describir el flujo del DAG del tema 12
- Actividad 4: generar el diagrama en draw.io con Mermaid
- Actividad 5: revisar y ajustar el gráfico final

## Material de esta sección

- [Ejercicio 01 - Instalar draw.io MCP y diagramar el flujo del DAG](./ejercicio-01-drawio-mcp-y-diagrama-dag.md)

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- instalar un servidor MCP en Claude Code
- entender cómo el agente usa herramientas MCP
- generar un diagrama técnico desde Mermaid
- representar visualmente el flujo del DAG del tema 12
