# Práctica - Módulo 08

## Objetivo práctico

Crear contexto persistente para agentes dentro de un repositorio real, usando `CLAUDE.md`, `MEMORY.md`, `AGENTS.md` y otros archivos complementarios.

## Modalidad sugerida

Laboratorio guiado en parejas o individual.

## Actividades

- Actividad 1: copiar el repositorio base a una cuenta personal
- Actividad 2: clonar la copia personal localmente
- Actividad 3: crear `CLAUDE.md` y `MEMORY.md`
- Actividad 4: crear `AGENTS.md` en Codex
- Actividad 5: identificar otros archivos útiles para contexto
- Actividad 6: practicar comandos principales de Claude Code y flujos equivalentes en Codex
- Actividad 7: usar Obsidian para organizar conocimiento generado por IA

## Criterio de éxito

Al terminar la práctica, el estudiante debe poder:

- copiar un repositorio real a una cuenta personal y clonarlo
- crear archivos de contexto iniciales
- explicar para qué sirve cada archivo
- usar comandos básicos de Claude Code relacionados con proyecto y memoria
- crear `AGENTS.md` para dar contexto operativo a Codex
- usar Obsidian para organizar ideas, prompts, decisiones y hallazgos técnicos
